# Grid Sequencer — Teratron Edition

ブラウザだけで動く、スマホ対応のステップシーケンサーです。
Web Audio API のみで音を生成しており、サーバー不要・インストール不要で動作します。

---

## 機能

### メロディグリッド
- **17音 × 16ステップ**（C4〜E5、半音含む）
- タップでセルをオン/オフ
- 現在再生中のステップがハイライト表示

### ドラムグリッド
- **KICK / TOM** の2トラック × 16ステップ
- シンセサイザーによるドラム音（Web Audio API 生成）

### コードセレクター
- メジャー、マイナー、7th、m7、maj7、sus など **25種類以上**
- 次のループ先頭で切り替わる（オレンジ = 待機中、黄色 = 演奏中）
- 停止中にタップするとプレビュー再生

### ミニキーボード
- **2オクターブ**（C4〜B5）の白鍵・黒鍵
- Rhodes 風エレクトリックピアノ音（サイン波 + 倍音 + アタッククリック）
- マルチタッチ対応：複数の指で和音を弾ける
- 指をスライドすると音が滑らかに移行

### ボイス（音色）
| 値 | 音色 |
|----|------|
| Flute | フルート風（倍音少なめ）|
| Strings | ストリングス風 |
| Choir | コーラス風 |

### コントロール
| 操作 | 内容 |
|------|------|
| PLAY / STOP | シーケンサーの再生・停止 |
| BPM スライダー | 40〜200 BPM |
| VOL スライダー | マスターボリューム |
| CLR | グリッドとドラムをクリア（再生停止） |

## 使い方

1. `index.html` をブラウザで開く
2. 画面をタップ（iOS/Android の AudioContext 解除）
3. グリッドのセルをタップしてパターンを組む
4. **PLAY** を押して再生
5. コードセレクターでコードを選ぶ（次のループから反映）
6. 下のミニキーボードでエレピを一緒に弾く

ローカルサーバーは不要です。ファイルをダブルクリックするだけで動きます。

## 技術仕様

- **言語**: バニラ JavaScript (ES2020) / HTML / CSS — フレームワーク・ライブラリなし
- **音声**: Web Audio API（`AudioContext`, `OscillatorNode`, `GainNode`, `ConvolverNode`）
- **リバーブ**: ホワイトノイズからインパルス応答を生成する合成リバーブ
- **レイアウト**: Flexbox、CSS カスタムプロパティで画面サイズに動的対応
- **スマホ対応**: `100dvh`、`env(safe-area-inset-*)` 対応、iOS Safari テスト済み

### ファイル構成

```
index.html   — マークアップ・レイアウト構造
style.css    — スタイル・アニメーション・レスポンシブ対応
app.js       — 全ロジック（シーケンサー・シンセ・UI）
```
